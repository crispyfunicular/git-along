Programmation et projet encadré

Mini-projets 1-2-3

Le miniprojet-3 (output = tableau .html avec Bulma) résulte du miniprojet-2 (output = tableau .tsv), qui résulte lui-même du miniprojet-1 (output directement dans le terminal).

Ces projets successifs avaient pour objectif de simuler une petite chaîne de collecte web. Le but était d'écrir en bash un script capable :

  1. d'extraire, pour chaque URL fourni dans une liste au format .txt :
    • des métadonnées (code HTTP, charset)
    • un comptage de mots

    En utilisant curl, lynx, wc, cut et grep

  2. puis de produire en sortie :
    • une sortie tabulaire pour le miniprojet-2
    • une page HTML utilisant le style Bulma avec un tableau lisible pour le miniprojet-3.
